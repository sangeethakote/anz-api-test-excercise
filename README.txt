Running in eclipse :
Import the existing maven project
Right click -> Run As -> Maven test

Running in command line :
Go to folder location anz-test-exercise
Type mvn test


Feature File : anz-api-test-excercise/src/test/resources/features/weather-station.featur

Step definition File : anz-api-test-excercise/src/test/java/stepdefinitions/WeatherStationSteps.java

Test runner File : anz-api-test-excercise/src/test/java/stepdefinitions/TestRunner.java

Results are saved as html report under anz-api-test-excercise/target/JunitReports/report.html
